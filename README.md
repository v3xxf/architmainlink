# Archit Kedia - Trading Profile

A modern, responsive trading profile website with animated trading elements.

## Features

- Real-time trading price ticker
- Animated trading charts and candlestick patterns
- Responsive design for mobile, tablet, and desktop
- Smooth animations and interactions
- Trading statistics display

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)

## Deployment

This project is deployed on Vercel.

## Local Development

Simply open `index.html` in your browser or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (http-server)
npx http-server
```

Then visit `http://localhost:8000`
